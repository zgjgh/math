# Project Instructions

- After each substantive user interaction in this workspace, update `chat_history.md` by running `python .\scripts\update_chat_history.py --output .\chat_history.md`.
- Commit the updated archive and relevant project changes to `main`, then push to `origin/main`, unless the user explicitly says not to push.
- Do not commit local cache files such as `__pycache__/`, `*.pyc`, LaTeX auxiliary files, or the Windows reserved `nul` path.
